# Uploading Files

Documents uploaded to a knowledge base folder are indexed for AI retrieval during chat sessions.

## Before You Upload

You need at least one knowledge base folder. If you have none, [create a folder first](manage.md#creating-a-folder).

## Uploading a Document

1. Open **Files** from the left sidebar
2. Click the knowledge base folder you want to add a file to
3. (Optional) Select or navigate to a subfolder
4. Click the **Upload** button (cloud/upload icon)
5. Select a file from your computer
6. A progress indicator shows upload status
7. The file appears in the document list when complete

## Upload Progress

While a file is uploading you will see a progress percentage. Do not navigate away until the upload is complete.

## Subfolder Organization

You can organize documents within a folder using subfolders. Subfolders appear as filter tabs at the top of the document list. To upload into a specific subfolder, select it before clicking **Upload**.

## File Size and Format

- Supported formats: PDF, DOCX, TXT, Markdown, and common text-based document types
- Files are ingested and indexed automatically after upload — this may take a few moments for large documents

## After Uploading

Once indexed, the document is available for AI retrieval. You can:

- [Download a document](manage.md#downloading-a-document) to get the original file back
- [Delete a document](manage.md#deleting-a-document) to remove it from the knowledge base
