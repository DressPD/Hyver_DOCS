# Managing Folders and Documents

## Creating a Folder

1. Open **Files** from the left sidebar
2. Click the **New Folder** button (+ icon)
3. Enter a **Name** (required)
4. Enter an optional **Description** to help identify the folder later
5. Click **Create**

The new folder appears at the top of the folder list and opens automatically.

## Deleting a Folder

> ⚠️ Deleting a folder removes all documents inside it. This action cannot be undone.

1. In the folder list, hover over the folder you want to delete
2. Click the **Delete** icon (trash icon)
3. Confirm the deletion in the confirmation dialog

## Searching Folders

Use the **Search** bar at the top of the Files page to filter folders by name or description. The list updates as you type.

## Downloading a Document

1. Open the knowledge base folder
2. Find the document in the list
3. Click the **Download** icon next to the document
4. The file opens in a new tab via a signed download URL

## Deleting a Document

1. Open the knowledge base folder
2. Find the document in the list
3. Click the **Delete** icon next to the document
4. Confirm the deletion in the confirmation dialog

The document is removed from the knowledge base and will no longer be available for AI retrieval.

## Storage and Usage

The **Files** overview page shows:

- **Total documents** – Combined document count across all folders
- **Total storage** – Combined size of all uploaded files in human-readable format (KB, MB, GB)

Each folder card also displays its individual document count and size.
