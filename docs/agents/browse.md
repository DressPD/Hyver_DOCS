# Browse Agents

Find and use agents from your collection and the marketplace.

## Accessing Agents

1. Click **Agents** in the navigation sidebar
2. The Agents page opens showing your agents and marketplace

## Page Sections

### Your Agents

Agents you own or can edit:
- Created by you
- Shared with you (with edit permissions)

### Marketplace

Public agents from the Hyver community:
- Curated selection
- Ready to use
- Various specializations

## Searching Agents

Use the search bar to find agents:

1. Click the **Search** field
2. Type keywords (name, description, capabilities)
3. Results filter in real-time

## Agent Cards

Each agent appears as a card showing:

- **Icon/Avatar** - Visual identifier
- **Name** - Agent's display name
- **Description** - Brief overview of capabilities
- **Chat** button - Start a conversation immediately
- **Configure** button - Edit settings (your agents only)

## Actions

### Start a Chat

1. Find an agent you want to use
2. Click **Chat**
3. A new chat opens with this agent selected
4. Start your conversation

### View Details

1. Click on an agent card
2. A modal opens with full details
3. See configuration, tools, and knowledge base info

### Quick Actions

Right-click (or long-press) agent cards for:
- Copy link
- Add to favorites
- Delete (your agents only)

## Filtering

Filter agents by:
- **Type** - Internal vs External
- **Category** - Domain/specialization
- **Tools** - Agents using specific tools

## Tips

- Save frequently used agents to favorites
- Check marketplace regularly for new agents
- Use search to quickly find specific capabilities
