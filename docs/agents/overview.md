# Agents Overview

Agents are specialized AI assistants with custom instructions, knowledge, and tool access.

## What is an Agent?

An agent is an AI personality customized for specific tasks:

- **Custom instructions** - Define how the agent behaves
- **Knowledge base** - Upload documents for the agent to reference
- **Tools** - Give the agent access to external services
- **Model selection** - Choose the underlying AI model

## Types of Agents

### Your Agents

Agents you've created or have edit access to:
- Fully customizable
- Can add/remove tools
- Can upload knowledge base documents
- Can share with others

### Marketplace Agents

Pre-built agents from the Hyver community:
- Ready to use immediately
- Read-only configuration
- Curated for quality

## Using Agents

### In Chat

1. Start a new chat
2. Type `@` and select an agent
3. Or click **Chat** on any agent card
4. The agent's personality and capabilities are now active

### Agent Capabilities

Depending on configuration, agents can:
- Answer questions from their knowledge base
- Use connected tools on your behalf
- Follow specific instruction patterns
- Maintain consistent personality

## Agent Cards

When browsing agents, each card shows:

| Element | Description |
|---------|-------------|
| **Name** | Agent's display name |
| **Description** | What the agent does |
| **Chat button** | Start a conversation |
| **Configure** | Edit settings (if owned) |
