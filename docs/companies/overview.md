# Companies Overview

Companies are multi-agent teams that work together to handle complex tasks.

## What is a Company?

A Company (also called a Swarm) is a group of agents that:

- **Collaborate** - Multiple agents handle different aspects
- **Route intelligently** - Tasks go to the right specialist
- **Coordinate** - An orchestrator manages the workflow

Think of it as assembling a team of specialists.

## Why Use Companies?

### Complex Tasks

Some tasks require multiple skills:
- Research + Writing + Review
- Data Analysis + Visualization + Summary
- Code + Testing + Documentation

### Specialization

Instead of one generalist agent:
- Each agent excels at one thing
- Routing ensures the right agent handles each part
- Better results through specialization

### Workflow

- Orchestrator agent manages flow
- Specialists focus on their expertise
- Handoffs happen automatically

## Types of Companies

### Your Companies

Companies you've created:
- Fully configurable
- Add/remove agents
- Set routing strategy

### Marketplace Companies

Pre-built company templates:
- Ready to use
- Tested configurations
- Good starting points

## Using Companies in Chat

1. Go to **Companies** page
2. Find a company
3. Click **Chat**
4. Your message goes to the orchestrator
5. Orchestrator routes to specialists as needed

## Company Components

| Component | Role |
|-----------|------|
| **Orchestrator** | Manages workflow, makes routing decisions |
| **Agents** | Specialists that handle specific tasks |
| **Routing Strategy** | Rules for how tasks are distributed |
