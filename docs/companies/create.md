# Create a Company

Build a multi-agent team for complex workflows.

## Starting Creation

1. Go to **Companies** in the navigation
2. Click **Create Company**
3. The creation form opens

## Basic Configuration

### Name

Give your company a clear name:
- "Research Team"
- "Content Pipeline"
- "Dev Squad"

### Description

Explain the company's purpose:
- What tasks it handles
- When to use it
- Expected outcomes

## Adding Agents

### Select Agents

1. Click **Add Agents**
2. Browse your agents and marketplace
3. Select agents to include
4. Click **Add**

### Agent Selection Tips

- Include specialists for each required skill
- Don't add too many - keep it focused
- Ensure agents complement each other
- Verify all agents have needed tools/credentials

## Orchestrator

The orchestrator agent manages the company:

### Selecting Orchestrator

1. Choose from your added agents
2. Or let Hyver auto-select
3. The orchestrator coordinates all other agents

### Orchestrator Role

- Receives incoming messages
- Decides which agent handles what
- Manages handoffs between agents
- Synthesizes final responses

1. Review configuration
2. Click **Create** or **Save**
3. Company appears in "Your Companies"
