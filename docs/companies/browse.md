# Browse Companies

Find and use multi-agent teams.

## Accessing Companies

1. Click **Companies** in the navigation sidebar
2. The Companies page opens

## Page Sections

### Your Companies

Companies you own or can edit:
- Created by you
- Shared with you

### Marketplace

Pre-built company configurations:
- Community templates
- Ready to use
- Various use cases

## Searching

Use the search bar to find companies:

1. Type keywords
2. Results filter in real-time

Search by:
- Company name
- Description
- Included agents
- Use case

## Company Cards

Each company shows:

- **Name** - Display name
- **Description** - What the company does
- **Agent count** - Number of agents in team
- **Chat** button - Start using the company
- **Configure** - Edit settings (if owned)

## Actions

### Start a Chat

1. Find a company
2. Click **Chat**
3. New chat opens with company active
4. Your messages go through the orchestrator

### View Details

Click a company card to see:
- Full description
- List of included agents
- Routing strategy
- Orchestrator info

## Tips

- Start with marketplace companies to learn patterns
- Note which agents are included
- Understand the routing strategy before using
