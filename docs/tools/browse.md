# Browse Tools

Explore available tools and integrations.

## Accessing Tools

1. Click **Tools** in the navigation sidebar
2. The Tools catalog opens

## Finding Tools

### Search

Use the search bar to find specific tools:
- Tool name
- Service name (Gmail, Slack, etc.)
- Capability

### Categories

Tools are organized by category:
- Communication (Slack, Email)
- Productivity (Google Workspace, Calendar)
- Development (GitHub, APIs)
- Custom integrations

## Tool Cards

Each tool shows:

| Element | Description |
|---------|-------------|
| **Icon** | Service logo |
| **Name** | Tool name |
| **Description** | What it does |
| **Status** | Connected / Not connected |
| **Chat** button | Start a chat (if connected) |
| **Connect** button | Set up credentials |

## Tool Status

### Connected ✓

- Credentials configured
- Ready to use
- Can add to agents

### Not Connected

- Needs credentials
- Click **Connect** to set up
- Cannot use until configured

## Actions

### Start a Chat

If a tool is connected:
1. Click **Chat**
2. New chat opens
3. Ask the AI to use this tool

### Connect Tool

If not connected:
1. Click **Connect**
2. Follow the setup flow
3. See [Connect Credentials](credentials.md)

### View Details

Click a tool card to see:
- Full description
- Required permissions
- Setup instructions
- Connected status
