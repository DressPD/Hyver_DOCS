# Create a Skill

Build a reusable prompt system for work you do often.

## Starting Creation

1. Go to **Skills** in the navigation
2. Click **Create new skill**
3. The creation form opens

## Basic Configuration

### Name

Give the skill a clear, specific name:
- "Code Review Assistant"
- "Weekly Update Writer"
- "Research Brief Builder"

### Description

Explain what the skill helps with:
- when to use it
- expected outcome
- what makes it different

### Category

Choose the category that best fits the workflow so the skill is easy to find later.

## Prompt Template

This is the core of the skill.

Include:
- the role or objective
- the structure you want in the answer
- any constraints or output format
- context that improves consistency

## Visibility

### Private

Only you can use and edit the skill.

### Public

The skill becomes available in the marketplace for others to launch.

## Saving

1. Review the name, description, category, and prompt
2. Click **Create** or **Save**
3. The skill appears in **Your Skills**

## Next Steps

After creating a skill:
- launch it in chat
- refine the prompt after real usage
- publish it when it is ready for wider reuse
