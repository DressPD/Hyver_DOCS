# First Steps in Hyver

After signing in, you'll land in the **Chat** workspace. Here's how to get started.

## The Welcome Screen

When you open a new chat, you'll see:

- A **welcome message** from Hyver
- **Quick-start suggestions** - agents, companies, or tools you can try immediately

## Start Your First Chat

### Option 1: Just Start Typing

1. Click in the message box at the bottom
2. Type your question or prompt
3. Press **Enter** or click **Send**

The AI will respond directly. This is the simplest way to use Hyver.

### Option 2: Use a Quick-Start Suggestion

1. Look at the suggestion pills on the welcome screen
2. Click one that interests you (e.g., an agent name)
3. That agent will be added to your chat
4. Type your message and send

### Option 3: Mention an Entity

While typing, you can mention agents, tools, or companies:

1. Type `@` in the message box
2. A dropdown appears with available entities
3. Select one to add it to your chat
4. The AI will use that entity to help answer

## Understanding the Interface

### Navigation (Left Sidebar)

- **Chat** - Your conversations
- **Tasks** - Running and completed jobs
- **Agents** - Browse and manage agents
- **Companies** - Multi-agent teams
- **Tools** - External integrations

### Chat Area (Center)

- **Message history** - Your conversation
- **Message box** - Type and send messages
- **Attachments** - Add files to your message

### Activity Pane (Right, Optional)

- Shows reasoning and activity during AI responses
- Toggle with the activity button
